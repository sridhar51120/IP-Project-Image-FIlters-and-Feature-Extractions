Spatial filtering is a common technique in image processing used to enhance or modify the appearance of an image by applying a filter or kernel to the image's pixel values. Two common types of spatial filtering are sharpening and smoothing.

Sharpening: Sharpening is used to enhance the edges and fine details in an image, making it appear more defined. It involves applying a high-pass filter to emphasize the differences in pixel values between neighboring pixels. The most common sharpening filter is the Laplacian filter or the Sobel filter. Smoothing: Smoothing, also known as blurring, is used to reduce noise in an image or to create a smoother appearance. It involves applying a low-pass filter to average or blur neighboring pixel values. The most common smoothing filter is the Gaussian filter.

import cv2 import numpy as np # Load an image image = cv2.imread('input_image.jpg') # Sharpening using Laplacian filter sharpened_image = cv2.Laplacian(image, cv2.CV_64F) sharpened_image = np.uint8(np.absolute(sharpened_image)) # Smoothing using Gaussian filter smoothed_image = cv2.GaussianBlur(image, (5, 5), 0) # Display the original, sharpened, and smoothed images cv2.imshow('Original Image', image) cv2.imshow('Sharpened Image', sharpened_image) cv2.imshow('Smoothed Image', smoothed_image) # Save the results cv2.imwrite('sharpened_image.jpg', sharpened_image) cv2.imwrite('smoothed_image.jpg', smoothed_image) # Wait for a key press and then close all windows cv2.waitKey(0) cv2.destroyAllWindows()
