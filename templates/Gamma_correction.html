Gamma correction

Definition

Gamma correction is a technique used in image processing to adjust the brightness of an image. It is often applied to compensate for the non-linearity of human perception of brightness. The gamma correction function maps the input pixel values to new values, making dark areas of the image lighter and bright areas darker.

Brief overview of how Gamma correction works
1. Gamma Value Selection:

You need to choose an appropriate gamma value. A gamma value less than 1 (e.g., 0.5) will brighten the image, while a value greater than 1 (e.g., 2.0) will darken it. A gamma of 1 will leave the image unchanged.


2. Apply Gamma Correction

For each pixel in the image, raise the original pixel value to the power of the gamma value.


3. Normalize

Optionally, you can normalize the pixel values to the desired output range (e.g., 0-255 for an 8-bit image) to ensure the corrected image maintains the same dynamic range as the original.


Code

Python Code
# Import Cv2 Library
import cv2
import numpy as np

# Load the image
image = cv2.imread('input_image.jpg')

# Define the gamma value (e.g., 1.5 for brightening, 0.5 for darkening)
gamma = 1.5

# Apply gamma correction
gamma_corrected = np.power(image/255.0, gamma)

# Normalize the pixel values to the 0-255 range
gamma_corrected = (gamma_corrected * 255).astype(np.uint8)

# Save the gamma-corrected image
cv2.imwrite('gamma_corrected_image.jpg', gamma_corrected)
